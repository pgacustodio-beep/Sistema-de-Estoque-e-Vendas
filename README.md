# Sistema de Estoque Java

Sistema desenvolvido em Java utilizando NetBeans.

## Funcionalidades
- Cadastro de produtos
- Controle de estoque
- Sistema de vendas
- Login de usuários
- Relatórios

## Tecnologias
- Java
- Swing
- MySQL

## Objetivo
Projeto desenvolvido para aprendizado e portfólio.
